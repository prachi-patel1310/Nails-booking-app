The Nails Booking App is a modern UI/UX design created to simplify salon appointment booking.
It focuses on providing a clean, aesthetic, and user-friendly experience for users who want to book nail services quickly and effortlessly.

This design is created entirely using Canva, showcasing how Canva can be used to build complete mobile UI screens.
